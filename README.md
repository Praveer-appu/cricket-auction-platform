# 🏏 Cricket Auction Platform

A web-based Cricket Auction Platform that digitizes and automates the player auction process with real-time bidding and budget management.

---

## 🚀 Features

- 🔥 Real-time bidding system
- 💰 Automatic budget deduction
- 🏆 Instant winner declaration
- 📊 Live player status (Available / Sold)
- 🛠 Admin dashboard controls
- ⚡ WebSocket-based live updates
- 🗄 MongoDB integration

---

## 🛠 Tech Stack

- Backend: Python (FastAPI)
- Database: MongoDB
- Frontend: HTML, CSS, JavaScript
- Real-time Communication: WebSockets
- Deployment Ready (Railway compatible)
