🏏 Cricket Auction Platform – AI Powered

An intelligent web-based Cricket Auction Platform that digitizes and automates the player auction process with real-time bidding, budget management, and AI-driven decision support.

🚀 Key Features

🔥 Real-time bidding system with WebSocket updates
💰 Automatic team budget management
🏆 Instant winner declaration
📊 Live player status (Available / Sold)
🛠 Admin dashboard with full auction control
🗄 MongoDB-based persistent data storage

🤖 AI-Powered Smart Auction Engine

🧠 Player Price Prediction using Machine Learning
📈 Data-driven valuation before bidding begins
⚠ Real-time Overbidding Detection System
💡 AI alerts to prevent budget mismanagement
📊 Predictive analytics integrated into auction flow

🛠 Tech Stack

Backend: Python (FastAPI)
Database: MongoDB
Frontend: HTML, CSS, JavaScript
Real-time Communication: WebSockets
Machine Learning: scikit-learn, NumPy
Deployment: Railway-compatible

📌 Project Purpose

To simulate a professional IPL-style cricket auction digitally while enhancing decision-making through AI-powered price prediction and intelligent bidding risk detection.
